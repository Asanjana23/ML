# Mental-Health-Detection-Using-Speech-Facial-Expressions
A multimodal AI application for emotion recognition using speech and facial expressions. Built with TensorFlow, Streamlit, OpenCV, Librosa, CNN, LSTM, and the Gemini API to deliver emotion analysis and personalized wellness recommendations.


## 🚀 Features

- Emotion detection from facial images using CNN
- Speech emotion recognition using LSTM and Librosa
- Multimodal emotion analysis
- Mood tracking and emotion visualization
- Personalized wellness recommendations using the Gemini API
- Interactive Streamlit web application

## ⚙️ Workflow

1. Upload a speech recording or facial image.
2. Extract speech and image features.
3. Predict emotions using CNN and LSTM models.
4. Display emotion analysis and mood insights.
5. Generate personalized wellness recommendations using the Gemini API.
